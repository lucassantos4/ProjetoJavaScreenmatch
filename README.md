# Projeto Java Screenmatch

Este repositório contém um **projeto de estudo em Java**, desenvolvido durante o aprendizado da **carreira Java da Alura**.
O objetivo do projeto é praticar conceitos fundamentais de **Programação Orientada a Objetos (POO)** e estruturação de aplicações em Java.

## Objetivo

Praticar e consolidar conhecimentos como:

* Criação e uso de **classes e objetos**
* **Encapsulamento**
* **Herança e polimorfismo**
* Manipulação de **listas e coleções**
* Organização de código em um projeto Java

## Sobre o projeto

O **Screenmatch** simula um sistema simples para organização e avaliação de conteúdos, permitindo aplicar na prática conceitos importantes da linguagem Java.

Este projeto faz parte do processo de aprendizado dentro da **carreira Java da Alura**, sendo utilizado exclusivamente para fins de **estudo e prática de programação**.

## Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)

## Observação

Este projeto foi desenvolvido com fins educacionais como parte dos estudos na **plataforma Alura**.
